# Age Eligibility Checker

A simple Python program that:

* Asks for the user's name
* Asks for the user's age
* Checks if the user is eligible (18 or older)
* Asks if the user wants to create an account
* Creates the account if the user enters "yes"

## Requirements

* Python 3

## How to Run

1. Download the file.
2. Open a terminal in the project folder.
3. Run:

```bash
python main.py
```

## Example

```
Enter your name: Imran
Enter your age: 20
Nice Imran, your eligible.

Do you want to create an account (yes,no): yes
Your account has been made.
```
